# Item Textures

This directory contains item texture PNG files for the resource pack:

- `iron_axe.png` - Iron axe texture
- `iron_sword.png` - Iron sword texture  
- `stone_axe.png` - Stone axe texture
- `wooden_axe.png` - Wooden axe texture
- `iron_door.png` - Iron door texture
- `lime_dye.png` - Lime dye texture
- `soul_lantern.png` - Soul lantern texture
- `oxidized_copper_lantern.png` - Oxidized copper lantern texture
- `honeycomb.png` - Honeycomb texture
- `wolf_spawn_egg.png` - Wolf spawn egg texture
